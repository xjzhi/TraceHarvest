# TraceHarvest - 攻防信息收集工具

基于多搜索引擎的自动化信息收集工具，通过组合关键词批量搜索并提取敏感信息（手机号、邮箱）。

## 功能特性

- **组合查询** — 主查询词 × 副查询词笛卡尔积，自动生成所有搜索组合
- **多引擎支持** — 百度、Bing、微信（搜狗），Google 待开发
- **敏感信息提取** — 正则匹配手机号（1[3-9]开头11位）、邮箱地址
- **翻页采集** — 可配置每引擎结果数和最大翻页数
- **百度频率控制** — 可配置请求间隔（默认120秒），交错调度避免封禁
- **结果导出** — 表格展示 + CSV 导出（UTF-8 with BOM）
- **白色科技风格 GUI** — 字体自适应窗口大小

## 工作流程

```
主查询词 × 副查询词 → 笛卡尔组合 → 多引擎搜索 → 翻页采集文章 → 提取手机号/邮箱 → 表格展示 → 导出CSV
```

**示例**：主词 `南瑞 英大`，副词 `招聘 招标 领导`，自动生成6组查询：

| 组合 | 查询 |
|------|------|
| 南瑞 × 招聘 | `南瑞 招聘` |
| 南瑞 × 招标 | `南瑞 招标` |
| 南瑞 × 领导 | `南瑞 领导` |
| 英大 × 招聘 | `英大 招聘` |
| 英大 × 招标 | `英大 招标` |
| 英大 × 领导 | `英大 领导` |

**百度交错调度**：选择百度+其他引擎时，百度首条立即搜索，其余引擎穿插执行，后续百度查询按设定间隔逐条执行，最大化利用等待时间。

## 安装

```bash
pip install -r requirements.txt
```

依赖：`requests`、`beautifulsoup4`、`lxml`

## 使用

```bash
python 信息收集工具.py
```

1. 输入目标组织/域名（主查询词，空格分隔多个）
2. 输入搜索关键字（副查询词，空格分隔多个）
3. 勾选搜索引擎（百度、Bing、微信）
4. 若勾选百度，扫描前设置请求间隔
5. 点击开始扫描
6. 查看结果表格，导出 CSV

## 打包为 EXE（可选）

```bash
pip install pyinstaller
pyinstaller --onefile --windowed 信息收集工具.py
```

## 注意事项

- 百度有反爬机制，建议请求间隔不低于120秒，频繁请求可能触发IP封禁（临时性，冷却后恢复）
- 微信搜狗搜索触发验证码时将自动停止该引擎扫描
- 本工具仅供授权安全测试与学习研究使用
